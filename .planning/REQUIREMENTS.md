# Requirements: v1.8 R Package Migration

**Milestone goal:** Convert ChemReg into a proper R package so the cleaning and curation pipeline can be used headlessly from scripts without launching the Shiny UI.

**Source:** R_PACKAGE_MIGRATION.md analysis + milestone scoping (2026-04-13)

---

## v1 Requirements

### Package Scaffolding

- [x] **PKG-01**: Developer can run `library(chemreg)` after `devtools::install()` without errors
- [x] **PKG-02**: `DESCRIPTION` file exists with correct Imports (dplyr, purrr, stringr, stringi, tibble, tidyr, readxl, readr, rio, janitor, rlang, tidyselect, fs, here, ComptoxR) and Suggests (shiny, bslib, bsicons, reactable, reactable.extras, shinyjs, writexl, testthat)
- [x] **PKG-03**: `NAMESPACE` is generated by roxygen2 (all exported functions have `@export` tags, `devtools::document()` succeeds)

### Source File Cleanup

- [x] **SRC-01**: `R/cleaning_pipeline.R` has no bare `library()` calls and all function calls use `::` notation or `@importFrom`
- [x] **SRC-02**: `R/cleaning_reference.R` has no bare `library()` calls and all function calls use `::` notation or `@importFrom`
- [x] **SRC-03**: `R/consensus.R` has no bare `library()` calls and all function calls use `::` notation or `@importFrom`
- [x] **SRC-04**: Package passes `devtools::check()` with no errors (warnings about non-standard files are acceptable)

### App Relocation

- [x] **APP-01**: Shiny app lives at `inst/app/app.R` (moved from project root)
- [x] **APP-02**: `R/run_app.R` exports `run_app()` that launches the app via `system.file("app", package = "chemreg")`
- [x] **APP-03**: Reference cache loaded via `system.file("extdata", "reference_cache", package = "chemreg")` — no `here::here()` calls remain in package source files
- [x] **APP-04**: `inst/app/app.R` has no `source()` loop (functions come from package namespace in installed mode; source loop only needed in dev mode)
- [x] **APP-05**: User can launch the app via `chemreg::run_app()` after package install

### Headless Pipeline

- [x] **HDL-01**: `curate_headless(input_path, output_path, tag_map, skip_flags)` is an exported function in the package
- [x] **HDL-02**: `curate_headless()` runs the full pipeline: file read → frontmatter detect → clean data → cleaning pipeline → dedup → CompTox search → consensus → XLSX export
- [x] **HDL-03**: `curate_headless()` produces a valid XLSX output when run against `uncurated_sswqs.csv`
- [x] **HDL-04**: `curate_headless()` returns a list with `data` and `audit_trail` invisibly

### Test Migration

- [x] **TST-01**: `tests/testthat/` directory structure exists with `tests/testthat.R` runner (`library(testthat); library(chemreg); test_check("chemreg")`)
- [x] **TST-02**: All existing test files renamed from `tests/test_*.R` → `tests/testthat/test-*.R` (dash convention)
- [x] **TST-03**: Pre-existing failure in `test_cleaning_reference.R` (expects 3 keys from `load_all_reference_lists`, gets 5: stop_words, block_patterns, functional_categories, strip_terms, isotope_lookup) is fixed
- [x] **TST-04**: `devtools::test()` passes with all tests green

## Out of Scope

*(Explicitly excluded from v1.8)*

- Converting to a CRAN-ready package (not the goal — internal/dev use)
- Splitting into multiple packages (chemreg.core + chemreg.shiny)
- Breaking changes to existing Shiny workflow or module APIs
