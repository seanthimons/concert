<!-- CompToxR functions used directly (no wrappers): -->

ComptoxR::ct_chemical_search_equal_bulk(query_list)
  - Takes character vector of chemical names
  - Returns data.frame with columns: dtxsid, preferredName, searchName, searchValue, rank
  - searchName indicates resolution method: "Approved Name", "CAS-RN", "Synonym"
  - One hit per input value

ComptoxR::ct_chemical_search_start_with(query)
  - Takes single character string
  - Returns data.frame with same columns as bulk
  - May return multiple matches per input

ComptoxR::as_cas(x)
  - Takes character vector of CAS-like strings
  - Returns character vector with normalized CAS (xx-yy-z format) or NA for invalid
  - Strips non-digits, removes leading zeros, reformats, validates check digit

ComptoxR::is_cas(x)
  - Takes character vector
  - Returns logical vector: TRUE if valid CAS format

Create R/prototype_pipeline.R implementing all 6 functions to make tests pass (GREEN phase).

**deduplicate_tagged_columns(df, tag_map):**
- For each entry in tag_map, extract the column values
- Split by tag type: "Name" columns → collect unique non-NA names, "CASRN" columns → collect unique non-NA CAS strings
- Build dedup_key_map: tibble with columns (row_idx, column_name, tag_type, dedup_key) where dedup_key is the original value
- Return list(unique_names=character_vector, unique_cas=character_vector, dedup_key_map=tibble)

**search_exact(unique_names):**
- If empty input, return empty tibble with correct columns
- Call `ComptoxR::ct_chemical_search_equal_bulk(unique_names)`
- Coerce rank to integer via `as.integer()`
- Select/rename to standard columns: searchValue, dtxsid, preferredName, searchName, rank
- `#NOTE: Takes lowest rank (top result) per searchValue — tweakable`

**search_starts_with(missed_names):**
- If empty input, return empty tibble
- Loop over each name, call `ComptoxR::ct_chemical_search_start_with(name)`
- Keep ALL results (multiple matches preserved per user decision — resolution deferred to downstream)
- Bind rows, add searchValue column = the input name that triggered the search
- Console message via `message()`: "Falling back on {N} misses with starts-with search..."

**validate_and_lookup_cas(unique_cas):**
- If empty input, return empty tibble
- Normalize with `ComptoxR::as_cas(unique_cas)`
- Check validity with `ComptoxR::is_cas()` on normalized values
- For valid CAS (non-NA after as_cas), lookup DTXSID via `ComptoxR::ct_chemical_search_equal_bulk(valid_cas)`
- Return tibble: original_cas, validated_cas, is_valid, dtxsid, preferredName

**run_tiered_search(dedup_result):**
- Message: "Searching {N} unique names..."
- Call search_exact(dedup_result$unique_names)
- Identify misses (NA dtxsid in results OR names not in results at all)
- If misses > 0: message "Falling back on {N} misses..." then call search_starts_with(missed_names)
- Message: "Validating {N} unique CAS numbers..."
- Call validate_and_lookup_cas(dedup_result$unique_cas)
- Combine all results into single tibble with a source_tier column ("exact", "starts_with", "cas")
- Include all-tier misses as rows with NA dtxsid

**map_results_to_rows(df, dedup_key_map, lookup_results):**
- Join dedup_key_map to lookup_results on dedup_key = searchValue (for names) and dedup_key = original_cas (for CAS)
- Join back to original df on row_idx
- Suffix lookup columns per tagged column to avoid name collisions (e.g., dtxsid_Chemical, dtxsid_cas)
- Return the full joined df
