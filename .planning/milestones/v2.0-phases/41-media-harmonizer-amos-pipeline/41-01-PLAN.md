**`harmonize_media(raw_media, orig_row_id)` (exported):**

Signature: `harmonize_media <- function(raw_media, orig_row_id = seq_along(raw_media))`

1. Empty-input guard: if `length(raw_media) == 0`, return 0-row tibble with all 6 columns typed correctly.

2. Load media table via `get_media_table()`. If NULL, return all-unmatched tibble.

3. Normalize input: `normalized <- trimws(tolower(raw_media))`

4. Build hash map for O(1) exact lookup: `lookup_hash <- stats::setNames(seq_len(nrow(media_tbl)), media_tbl$term)`

5. Exact match: `match_idx <- lookup_hash[normalized]`

6. For unmatched rows, attempt parent-walk via `walk_parent()`:
   - `walk_parent(term, media_tbl)` searches for the term as a substring within media_tbl$term entries, OR searches for media_tbl$term entries that are substrings of the input term
   - If a match is found, walk up the `parent` column to find the entry with a non-NA `media_category`
   - Return the matched row index or NA

7. Assign flags:
   - Exact match: `media_flag = ""`
   - Parent-walk match: `media_flag = "parent_walk"`
   - No match or NA input: `media_flag = "media_unmatched"`

8. Build output tibble:
```r
tibble::tibble(
  orig_row_id     = as.integer(orig_row_id),
```
