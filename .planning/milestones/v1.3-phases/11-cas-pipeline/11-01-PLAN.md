    Add 4 new functions to R/cleaning_pipeline.R after existing functions:

    1. **inject_row_lineage(df)**: Add `original_row_id = 1:nrow(df)` as first column via `dplyr::mutate(.before = 1)`. Simple one-liner.

    2. **normalize_cas_fields(df, tag_map)**:
       - Get CASRN columns from tag_map: `cas_cols <- names(tag_map)[tag_map == "CASRN"]`
       - If no CAS columns, return list(cleaned_data = df, audit_trail = empty tibble) — skip gracefully
       - Save df_before, apply `ComptoxR::as_cas()` to each CASRN column via `dplyr::mutate(across(all_of(cas_cols), ~ComptoxR::as_cas(.x)))`
       - Build audit trail with step_name = "normalize_cas" using existing build_audit_trail()
       - NOTE: build_audit_trail compares character columns only. as_cas() returns character. But need to handle NA comparison: values that were non-NA text and became NA are changes. Adjust comparison to treat original non-NA becoming new NA as a change. Use `!is.na(original) & is.na(cleaned)` as additional change condition in a custom audit for this step, OR convert NAs to a sentinel like "[NA]" before comparison. Simplest: write a CAS-specific audit that handles NA transitions.
       - Return list(cleaned_data, audit_trail)

    3. **rescue_cas_from_text(df, tag_map)**:
       - Get non-CASRN tagged columns: `non_cas_cols <- names(tag_map)[tag_map != "CASRN"]`
       - If none, return list(cleaned_data = df, audit_trail = empty tibble, new_tags = list())
       - For each non-CASRN column: call `ComptoxR::extract_cas(df[[col_name]])`
       - If any non-NA results: create `cas_extract_{col_name}` column, add to new_tags as "CASRN"
       - Strip CAS pattern from source text: `stringr::str_remove_all(x, "\\s*[\\(\\[]?\\d{1,7}-\\d{2}-\\d[\\)\\]]?\\s*")` then `str_squish()`
       - Build per-row audit entries for extractions (step = "rescue_cas")
       - Return list(cleaned_data, audit_trail, new_tags)

    4. **detect_multi_cas(df, tag_map)**:
       - Get all CASRN columns (including any cas_extract_* that were added)
       - `rowSums(!is.na(df[, cas_cols, drop = FALSE]))` for count
       - Add `multi_cas` (logical) and `multi_cas_count` (integer) columns
       - Return df (no audit trail needed — flagging, not transformation)

    5. **Update run_cleaning_pipeline(df, tag_map = NULL, reference_lists = NULL)**:
       - Add tag_map parameter (default NULL for backward compatibility)
       - First: inject_row_lineage(df)

       - If tag_map is not NULL:
         - normalize_cas_fields(df_after_trim, tag_map)
         - rescue_cas_from_text(df_after_cas_norm, tag_map) — use tag_map from normalize step
         - Merge rescue new_tags into tag_map for detect_multi_cas
         - detect_multi_cas(df_after_rescue, updated_tag_map)
       - Combine all audit trails
       - Return list(cleaned_data, audit_trail, new_tags) — new_tags is empty list if no tag_map
