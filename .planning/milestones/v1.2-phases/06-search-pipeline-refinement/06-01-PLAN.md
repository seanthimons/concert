<objective>
Reorder the search tier chain from exact → starts-with → CAS to exact → CAS → starts-with, and expand "Other" tagged columns to participate in the full search pipeline.

Purpose: CAS validation before starts-with increases precision for CAS-identifiable chemicals. Other tag inclusion allows users to curate arbitrary identifier columns (synonyms, supplier codes) alongside Name/CASRN.

Output: Modified R/curation.R with reordered tiers, 3-char minimum filter on starts-with, Other tag extraction, and corresponding tests.
</objective>
