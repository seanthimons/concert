## Decisions Made

1. **ECOTOX row count:** Documentation said 706 rows but actual unit_conversion table has 1418 unique entries after deduplication - used full dataset
2. **Molarity handling (D-14):** All molarity entries (M, mM, uM, nM, pM, mol/L, etc.) have:
   - to_unit = "M" (not mg/L)
   - multiplier = NA_real_ (not numeric)
   - confidence = "NEEDS_MW"
3. **Heavy dependencies:** DBI, duckdb, and arrow packages are NOT added to DESCRIPTION Imports - they're only needed for one-time data builds, documented as manual install requirement
4. **Synonym table design:** Used exact string matching only (no regex) - simpler implementation with sufficient coverage for known variants
