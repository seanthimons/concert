<objective>
Wire duration harmonization into both pipeline entry points (Shiny mod_harmonize.R and headless curate_headless.R) so that DurationUnit-tagged columns are routed through `harmonize_units(category="duration")` and the output populates `study_duration_value` and `study_duration_units` in the ToxVal schema.

Purpose: This completes the Phase 39 goal -- users can tag columns as Duration/DurationUnit and get harmonized duration values in the ToxVal export. The `map_to_toxval_schema()` function already reads `study_duration_value` and `study_duration_units` via `safe_extract_num()`/`safe_extract_char()`, so the only work is merging duration results into the curated data before the mapper is called.

Output: Both pipeline paths gain a duration harmonization stage; ToxVal exports contain populated study_duration fields when duration columns are tagged.
</objective>
