**Section 3: EXPAND PARENTHETICAL COMPOUND TERMS (D-04)**
Handle patterns like "water (freshwater)", "sediment (marine)":
```r
# Expand "X (Y)" -> c("X", "Y", "Y X")
expand_parenthetical <- function(term) {
  m <- stringr::str_match(term, "^(.+?)\\s*\\((.+?)\\)$")
  if (is.na(m[1, 1])) return(term)
  base <- trimws(m[1, 2])
  qualifier <- trimws(m[1, 3])
  c(base, qualifier, paste(qualifier, base))
}
```

Apply to all extracted terms, then deduplicate.

**Section 4: MATCH AGAINST CURATED ENVO SUBSET**

- If exact match in curated_media$term: mark as "matched"
- If no match: mark as "unmatched" (coverage gap)

Build a tibble of AMOS-derived terms that matched. For unmatched terms, attempt fuzzy assignment:
- If term contains a curated term as substring (e.g., "river water" contains "water"), assign the curated term's parent and media_category

**Section 7: REFRESH FUNCTION (AMOS-03)**
Define `refresh_amos_cache()` as a wrapper that re-runs the pipeline with staleness warning:
```r
refresh_amos_cache <- function(force = FALSE) {
  if (file.exists(curated_path)) {
    existing <- readRDS(curated_path)
    ts <- existing$fetch_timestamp[1]
    if (!is.na(ts) && !force) {
      age_days <- as.numeric(difftime(Sys.time(), as.POSIXct(ts), units = "days"))
      if (age_days < 30) {
        message(sprintf("Cache is %.0f days old (< 30). Use force=TRUE to refresh.", age_days))
        return(invisible(NULL))
      }
      message(sprintf("Cache is %.0f days old. Refreshing...", age_days))
    }
  }
  source(file.path(CHEMREG_ROOT, "scripts", "build_amos_media.R"))
}
```
