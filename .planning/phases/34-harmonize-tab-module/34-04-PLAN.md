# Plan 34-04: Fix Cascade Reset UX

**Gap:** Test 11 — Adding unit mapping resets harmonization state, requiring 2+ min re-run. Bad UX for iterative editing.

**Root Cause:** `observeEvent(data_store$unit_map_working)` at line 940 immediately sets `harmonize_results <- NULL` on any change.

## Approach: Stale Results Pattern + Incremental Re-harmonization

Two-part fix:
1. **Stale flag** — Don't clear results on edit; show warning, let user batch edits
2. **Incremental re-run** — Only re-process rows with changed/added units, not entire dataset

observeEvent(data_store$unit_map_working, {
  if (!is.null(prev_unit_map()) &&
      !identical(prev_unit_map(), data_store$unit_map_working)) {
    if (!is.null(data_store$harmonize_results)) {

### Task 7: Implement incremental re-harmonization

**File:** `R/mod_harmonize.R`

Modify "Run Harmonization" observer to support incremental mode:

```r
observeEvent(input$run_harmonize, {
  req(data_store$cleaned_data, data_store$numeric_tags)

  data_store$harmonize_results_stale <- FALSE

  # Determine if incremental or full run
  incremental <- length(data_store$changed_units) > 0 &&
                 !is.null(data_store$harmonize_results)

  if (incremental) {
    # Only re-process rows with changed units
    affected_rows <- which(data_store$cleaned_data[[unit_col]] %in% data_store$changed_units)

    if (length(affected_rows) > 0) {
      # Run harmonization on subset
      incremental_result <- harmonize_units(
        values = data_store$cleaned_data[[result_col]][affected_rows],
        units = data_store$cleaned_data[[unit_col]][affected_rows],
        unit_map = data_store$unit_map_working
      )

      # Merge back into full results
      data_store$harmonize_results[affected_rows, ] <- incremental_result
    }

    data_store$changed_units <- character(0)  # Clear tracking

  } else {
    # Full harmonization (first run or major changes)
    data_store$harmonize_results <- harmonize_units(...)
  }
})
```

### Task 10: Hash-based unit lookup + vectorized assignment

**File:** `R/unit_harmonizer.R`

Replace per-row `match()` with hash lookup and vectorized assignment:

**Before:**
```r
for (i in seq_len(n)) {
  idx <- match(norm_unit, unit_map$from_unit)  # O(m) per row = O(n*m) total
  if (!is.na(idx)) {
    harmonized_unit[i] <- unit_map$to_unit[idx]
    conversion_factor[i] <- unit_map$multiplier[idx]
  }
}
# Plus redundant O(n²) sapply at line 383
standard_rows <- !(molarity_mask | sapply(seq_len(n), ...))
```

**After:**
```r
# Build hash map once: from_unit -> row index (O(m))
lookup_hash <- setNames(seq_len(nrow(unit_map)), unit_map$from_unit)
lookup_hash_ci <- setNames(seq_len(nrow(unit_map)), tolower(unit_map$from_unit))

# Vectorized lookup for all standard rows at once (O(n))
std_units <- normalized[standard_mask]
lookup_idx <- lookup_hash[std_units]

# Case-insensitive fallback for unmatched
unmatched <- is.na(lookup_idx)
if (any(unmatched)) {
  lookup_idx[unmatched] <- lookup_hash_ci[tolower(std_units[unmatched])]
  unit_flag[standard_mask][!is.na(lookup_idx) & unmatched] <- "case_fallback"
}

# Vectorized assignment (O(n))
matched <- !is.na(lookup_idx)
harmonized_unit[standard_mask][matched] <- unit_map$to_unit[lookup_idx[matched]]
conversion_factor[standard_mask][matched] <- unit_map$multiplier[lookup_idx[matched]]
harmonized_unit[standard_mask][!matched] <- orig_unit[standard_mask][!matched]
conversion_factor[standard_mask][!matched] <- 1
unit_flag[standard_mask][!matched] <- "unmatched"

# Vectorized value computation (O(n))
harmonized_value[standard_mask] <- values[standard_mask] * conversion_factor[standard_mask]

# Handle molarity rows (vectorized where possible)
mol_with_mw <- molarity_mask & !is.na(mw_vec) & mw_vec > 0
mol_scales <- vapply(normalized[mol_with_mw], get_molarity_scale, numeric(1))
harmonized_value[mol_with_mw] <- values[mol_with_mw] * mw_vec[mol_with_mw] * mol_scales
harmonized_unit[mol_with_mw] <- "mg/L"
conversion_factor[mol_with_mw] <- mw_vec[mol_with_mw] * mol_scales

# Handle ppx rows (vectorized)
ppx_factors <- vapply(normalized[ppx_mask], get_ppx_conversion_factor, numeric(1))
ppx_targets <- vapply(seq_len(sum(ppx_mask)), function(i) {
  get_media_target(normalized[ppx_mask][i], media_vec[ppx_mask][i]) %||% "mg/L"
}, character(1))
harmonized_value[ppx_mask] <- values[ppx_mask] * ppx_factors
harmonized_unit[ppx_mask] <- ppx_targets
conversion_factor[ppx_mask] <- ppx_factors
```

**Performance impact:**
- Current: ~8 sec for 128k rows (per-row loop + repeated match)
- After: <1 sec for 128k rows (vectorized operations + hash lookup)
