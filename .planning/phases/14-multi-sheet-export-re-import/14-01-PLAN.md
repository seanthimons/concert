  <action>
    Create R/export_helpers.R with two functions:

    1. `build_export_sheets(raw, resolution_state, consensus_summary, cleaning_audit, reference_lists, column_tags, detection, file_info)` — accepts individual data store components (not the reactive object itself, for testability). Builds all 7 sheet data frames:
       - "Raw Data": raw as-is
       - "Curated Data": resolution_state with needs_review flag added, .pinned/.manual_entry removed (replicate existing logic from mod_review_results.R:788-793)
       - "Summary": consensus statistics tibble (replicate existing logic from mod_review_results.R:796-819)
       - "Cleaning Audit": cleaning_audit as-is (may be NULL — use empty tibble if so)
       - "Reference Lists": bind_rows of all reference list tibbles with type column (singular values)
       - "Column Tags": tibble with Column and Type columns from named list
       - "Pipeline Config": key-value tibble with chemreg_export=true, app_version, export_timestamp, detection_method, detection_confidence, file_name, file_size_bytes, pipeline_steps (unique steps from cleaning_audit)
       Returns the named list ready for writexl::write_xlsx().

    2. `validate_excel_size(df, sheet_name)` — checks nrow < 1,048,576 and ncol < 16,384. Throws informative error with sheet name and format(n, big.mark=",") if exceeded. Returns invisible(TRUE) on success.

    Create R/config_import.R with two functions:

    1. `parse_chemreg_export(file_path)` — two-stage validation:
       (a) readxl::excel_sheets() to check "Pipeline Config" exists
       (b) readxl::read_excel(sheet="Pipeline Config") to check chemreg_export=true
       If valid, also reads "Reference Lists" and "Column Tags" sheets.
       Returns list(reference_lists=df, column_tags=df, config=df) or NULL.
       Wrap in tryCatch for malformed files — return NULL on any error with warning().

    2. `merge_reference_lists(existing_lists, imported_ref_df)` — takes existing_lists (list of 3 tibbles keyed by plural names) and imported_ref_df (combined tibble with type column, singular values). For each type, filters imported_ref_df, sets source="imported", appends to existing list, deduplicates by term (imported wins via .keep_all=TRUE on the bind_rows order where imported comes last then distinct keeps first — so put imported FIRST in bind_rows to win). Returns updated list.
       NOTE on dedup: dplyr::distinct(term, .keep_all=TRUE) keeps the FIRST occurrence. To make imported win on conflicts, put imported rows first: bind_rows(imported_subset, existing) then distinct.
