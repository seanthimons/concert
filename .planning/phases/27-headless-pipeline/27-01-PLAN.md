Function body structure — wrap entire pipeline in an inner `pipeline()` function, then dispatch based on `verbose` (per D-01, using `withCallingHandlers` pattern from research):

```r
if (verbose) {
  pipeline()
} else {
  withCallingHandlers(
    pipeline(),
    message = function(m) invokeRestart("muffleMessage")
  )
}
```

Inside `pipeline()`, implement these steps in order:

**Step 1 — Validate input (per D-04, fail fast):**
- `if (!file.exists(input_path)) stop(sprintf("curate_headless: file not found: %s", input_path))`
- `file_ext <- tolower(tools::file_ext(input_path))`
- `if (!file_ext %in% c("csv", "xlsx", "xls")) stop(sprintf("curate_headless: unsupported file type '%s'. Use csv, xlsx, or xls.", file_ext))`

**Step 2 — Load reference lists (per D-02):**
- When `reference_lists` is NULL: `cache_dir <- system.file("extdata", "reference_cache", package = "chemreg")` then `reference_lists <- load_all_reference_lists(cache_dir)`
- When provided: validate keys with `expected <- c("stop_words", "functional_categories", "block_patterns", "strip_terms", "isotope_lookup")`, check `setdiff(expected, names(reference_lists))`, stop if missing

**Step 3 — Read file:**
- `message(sprintf("[headless] Reading file: %s", basename(input_path)))`
- `raw_df <- safely_read_file(input_path, file_ext)`
- Do NOT call `validate_file()` — it expects Shiny fileInput objects

**Step 4 — Detect frontmatter (per D-03):**
- When `header_row` is not NULL: `detection <- detect_data_start(raw_df, mode = "manual", manual_row = header_row)`
- When NULL: `detection <- detect_data_start(raw_df, mode = "auto")`
- `message(sprintf("[headless] Detection: method=%s, confidence=%.2f, header_row=%d", detection$method, detection$confidence, detection$header_row))`

**Step 5 — Extract and post-process:**
- `clean_data <- extract_clean_data(raw_df, detection)`
- `clean_data <- handle_merged_cells(clean_data)`
- `clean_data <- janitor::clean_names(clean_data)`
- `clean_data <- janitor::remove_empty(clean_data, which = c("rows", "cols"))`

**Step 6 — Validate tag_map against cleaned column names:**
- `missing_cols <- setdiff(names(tag_map), names(clean_data))`
- If any missing, `stop(sprintf("curate_headless: tag_map keys not found after column normalization: %s\nActual columns: %s", paste(missing_cols, collapse = ", "), paste(names(clean_data), collapse = ", ")))`

**Step 7 — Run cleaning pipeline:**
- `message("[headless] Running cleaning pipeline...")`
- `cleaning_result <- run_cleaning_pipeline(clean_data, tag_map, reference_lists)`
- `merged_tags <- c(tag_map, cleaning_result$new_tags)`

**Step 8 — Run curation pipeline:**
- `message("[headless] Running curation pipeline (CompTox API search)...")`
- `pipeline_result <- run_curation_pipeline(cleaning_result$cleaned_data, merged_tags)`
- `resolution_state <- pipeline_result$results`

Note on `skip_flags`: The `run_curation_pipeline()` function hardcodes `skip_flags = "isotope_match"` internally at its `deduplicate_tagged_columns()` call (line 561 of R/curation.R). The `skip_flags` parameter in the `curate_headless()` signature is accepted for forward-compatibility but is not currently forwarded. Add a comment: `# skip_flags reserved for future use; isotope_match skip is handled internally by run_curation_pipeline()`.

**Step 9 — Build export sheets and write XLSX:**
- Construct `file_info`: `file_info <- list(name = basename(input_path), size = file.info(input_path)$size)`
- Call `build_export_sheets(raw = raw_df, resolution_state = resolution_state, consensus_summary = pipeline_result$consensus_summary, cleaning_audit = cleaning_result$audit_trail, reference_lists = reference_lists, column_tags = merged_tags, detection = detection, file_info = file_info)`
- `fs::dir_create(dirname(output_path), recurse = TRUE)` (handles missing parent dirs)
- `writexl::write_xlsx(sheets, output_path)`
- `message(sprintf("[headless] Output written to: %s", output_path))`

**Step 10 — Return invisibly (per D-05):**
- `invisible(list(data = resolution_state, audit_trail = cleaning_result$audit_trail))`

**3. Regenerate NAMESPACE:**
Run `devtools::document()` to regenerate NAMESPACE from roxygen tags. Verify `export(curate_headless)` appears in NAMESPACE.
