```r
#' Match chemical names against WQX Characteristic Name dictionary
#'
#' Runs a three-tier lookup: (1) exact canonical, (2) alias crosswalk,
#' (3) Jaro-Winkler fuzzy against canonical names only. Returns a tibble
#' with one row per input name.
#'
#' @param names Character vector of analyte names to match
#' @param dictionary Tibble from load_wqx_dictionary() with columns: name, canonical_name, type
#' @param threshold Numeric similarity threshold for fuzzy acceptance (default 0.85).
#'   Internally converted to JW distance cutoff: distance <= (1 - threshold).
#' @param verbose Logical. If TRUE, emits per-name cli output (default FALSE).
#' @return Tibble with columns: input_name, wqx_name, match_tier, match_distance, alias_type
#' @export
```

Function signature (per D-06):
```r
match_wqx <- function(names, dictionary, threshold = 0.85, verbose = FALSE) {
```

Implementation body:

1. **Empty-input guard** — if `length(names) == 0`, return a zero-row tibble with the 5 correct columns typed as `character(0)`, `character(0)`, `character(0)`, `numeric(0)`, `character(0)`.

2. **Pre-allocate result vectors** — create 5 vectors of length `length(names)`, initialized: `input_name = names`, `wqx_name = NA_character_`, `match_tier = "none"`, `match_distance = NA_real_`, `alias_type = NA_character_`.

3. **Identify valid inputs** — `valid_idx <- which(!is.na(names) & trimws(names) != "")`. Only these indices proceed through tiers. Invalid indices keep their default "none"/NA values.

4. **Normalize once** — `names_clean <- tolower(trimws(names))` (per discretion: tolower once upfront).

5. **Build hash tables from dictionary** (per D-08):
   - Filter canonical rows: `canonical_rows <- dictionary[dictionary$type == "canonical", ]`
   - Filter alias rows: `alias_rows <- dictionary[dictionary$type %in% c("synonym", "standardize", "retired"), ]`
   - Deduplicate alias keys (per RESEARCH.md Pitfall 2): `alias_rows <- dplyr::distinct(dplyr::mutate(alias_rows, .lower_name = tolower(name)), .lower_name, .keep_all = TRUE)`
   - Build tier1_map: `stats::setNames(canonical_rows$name, tolower(canonical_rows$name))` — values are original-cased canonical names, keys are lowercased
   - Build tier2_map: `stats::setNames(alias_rows$canonical_name, tolower(alias_rows$name))` — values are canonical names, keys are lowercased alias names
   - Build tier2_type_map: `stats::setNames(alias_rows$type, tolower(alias_rows$name))` — values are "synonym"/"standardize"/"retired"

6. **Tier 1: Exact canonical** (MATCH-01):
   - `tier1_hits <- tier1_map[names_clean[valid_idx]]`
   - Where not NA: set `wqx_name[valid_idx] = tier1_hits`, `match_tier[valid_idx] = "exact"`, `match_distance[valid_idx] = NA_real_`, `alias_type[valid_idx] = NA_character_`
   - Track: `tier1_resolved <- valid_idx[!is.na(tier1_hits)]`, `unresolved_idx <- valid_idx[is.na(tier1_hits)]`

7. **Tier 2: Alias lookup** (MATCH-02):
   - `tier2_hits <- tier2_map[names_clean[unresolved_idx]]`
   - `tier2_types <- tier2_type_map[names_clean[unresolved_idx]]`
   - Where not NA: set `wqx_name`, `match_tier = "alias"`, `alias_type = tier2_types`
   - Track: `tier2_resolved`, `still_unresolved_idx`

8. **Tier 3: Fuzzy match** (MATCH-03, per D-01/D-02):
   - Only run if `length(still_unresolved_idx) > 0`
   - Build `canonical_name_vec <- canonical_rows$name` (23,304 names — fuzzy target)
   - `dist_matrix <- stringdist::stringdistmatrix(names_clean[still_unresolved_idx], tolower(canonical_name_vec), method = "jw")`
   - `best_dist <- apply(dist_matrix, 1, min)`
   - `best_idx <- apply(dist_matrix, 1, which.min)`
   - `best_match <- canonical_name_vec[best_idx]`
   - Distance cutoff: `accepted <- best_dist <= (1 - threshold)` — comment: `# JW distance: 0=identical, cutoff = 1 - threshold`
   - For accepted: set `wqx_name = best_match`, `match_tier = "fuzzy"`, `match_distance = best_dist`
   - For rejected: keep `match_tier = "none"`, `wqx_name = NA`, set `match_distance = best_dist` (nearest candidate distance shown per D-02)

9. **Verbose per-name logging** (MATCH-04, per D-05):
   - Only if `verbose == TRUE`, iterate over `valid_idx` and emit:
     - For "exact": `cli::cli_alert_success("'{input_name[i]}' -> '{wqx_name[i]}' [exact]")`
     - For "alias": `cli::cli_alert_success("'{input_name[i]}' -> '{wqx_name[i]}' [alias/{alias_type[i]}]")`
     - For "fuzzy": `cli::cli_alert_success("'{input_name[i]}' -> '{wqx_name[i]}' [fuzzy, dist={sprintf('%.3f', match_distance[i])}]")`
     - For "none": `cli::cli_alert_warning("'{input_name[i]}' -> unresolved (nearest: '{best_match_for_idx}', dist={sprintf('%.3f', match_distance[i])})")`
   - Note: for "none" verbose output, the nearest candidate name needs to be available. Store it in a helper vector during Tier 3 (`nearest_candidate` vector, set for still_unresolved_idx where not accepted).

10. **Summary logging** (always, per D-05):
    - Count tiers: `n_exact <- sum(match_tier == "exact")`, etc.
    - `cli::cli_inform(c("v" = "WQX match complete: {n_exact} exact, {n_alias} alias, {n_fuzzy} fuzzy, {n_none} unresolved"))`

11. **Return tibble** (per D-07):
    - `tibble::tibble(input_name = names, wqx_name = wqx_name, match_tier = match_tier, match_distance = match_distance, alias_type = alias_type)`
