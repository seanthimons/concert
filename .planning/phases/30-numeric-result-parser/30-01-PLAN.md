    Create file with these internal helpers and one exported function:

    1. `normalize_numeric_string(x)` (internal, not exported):
       - Input: character vector
       - Chain order: (a) replace unicode qualifiers ≥→>=, ≤→<= (b) replace `x10^` and `X10^` with `e` (c) detect Fortran exponents: pattern `(\d)([+-]\d+)$` where no `e/E` precedes → insert `e` (d) strip commas between digits (e) squish whitespace
       - Return: character vector of normalized strings

    2. `extract_qualifier(x)` (internal, not exported):
       - Input: character vector (already normalized)
       - Regex: `^\s*(<=|>=|<|>|~|=)\s*(.*)$` — greedy qualifier match, longest first
       - Return: list with two character vectors: `qualifier` and `remainder`
       - Per D-07: no match → qualifier = ""

    3. `detect_narrative(x)` (internal, not exported):
       - Input: character vector (remainder after qualifier extraction)
       - Known narratives (case-insensitive): "BDL", "ND", "non-detect", "nondetect", "trace", "not detected", "below detection", "below quantitation", "BQL", "LOD", "LOQ"
       - Also flag: empty string "", NA, whitespace-only
       - Return: logical vector, TRUE = narrative

       - Input: character vector of raw result strings
