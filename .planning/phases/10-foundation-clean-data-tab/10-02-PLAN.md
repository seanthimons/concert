  <name>Task 1: Create Clean Data module</name>
  <files>R/modules/mod_clean_data.R</files>
  <action>
    Create `R/modules/mod_clean_data.R` following the mod_run_curation.R pattern exactly.

    **mod_clean_data_ui(id):**
    - `ns <- NS(id)` for namespacing
    - Two conditionalPanels based on `output[ns("has_data")]`:
      - **When has_data is TRUE:**
        - `actionButton(ns("run_cleaning"), "Run Cleaning", class = "btn-success btn-lg", icon = icon("broom"))` — starts enabled (cleaning always re-runnable per user decision)
        - `uiOutput(ns("cleaning_summary"))` — text summary above table
        - `DT::dataTableOutput(ns("cleaned_table"))` — cleaned data display
      - **When has_data is FALSE:**
        - Empty state: centered div with `bsicons::bs_icon("broom", size = "3em")`, `h4("No data loaded")`, `p("Upload a file and it will appear here for cleaning.")`

    **mod_clean_data_server(id, data_store, on_cleaning_complete = NULL):**
    - `output$has_data` — reactive returning `!is.null(data_store$clean)`, with `outputOptions(suspendWhenHidden = FALSE)`
    - `observeEvent(input$run_cleaning, { ... })`:
      1. `req(data_store$clean)`
      2. Disable button with `shinyjs::disable("run_cleaning")`
      3. Wrap in `tryCatch()` with error notification on failure and `finally` to re-enable button
      4. Use `withProgress(message = "Cleaning data...", value = 0, { ... })`:
         - `incProgress(0.3, detail = "Converting unicode to ASCII")`
         - `incProgress(0.6, detail = "Trimming whitespace and punctuation")`
         - Call `run_cleaning_pipeline(data_store$clean)` — single call, pipeline handles steps internally
         - `incProgress(1.0, detail = "Complete")`
      5. Store results: `data_store$cleaned_data <- result$cleaned_data`, `data_store$cleaning_audit <- result$audit_trail`
      6. Show notification: `sprintf("Cleaning complete: %d transformations applied", nrow(result$audit_trail))`
      7. Call `on_cleaning_complete()` callback if not NULL
      8. Re-enable button in `finally` block
    - `output$cleaning_summary` — `renderUI()`:
      - `req(data_store$cleaning_audit, data_store$cleaned_data)`
      - Count unicode changes: `sum(data_store$cleaning_audit$step == "unicode_to_ascii")`
      - Count trim changes: `sum(data_store$cleaning_audit$step == "trim_whitespace_punctuation")`
      - Display: `div(class = "alert alert-info mb-3", p(sprintf("X rows cleaned, Y unicode chars fixed, Z fields trimmed")))` with actual counts
      - If 0 total changes: "No transformations needed — data is already clean"
    - `output$cleaned_table` — `DT::renderDataTable()`:
      - `req(data_store$cleaned_data)`
      - `DT::datatable(data_store$cleaned_data, options = list(pageLength = 25, scrollX = TRUE), rownames = FALSE)`
  </action>
